# SS4：生平多维叙事剧本

**Skill ID**：`SS4`  
**核心定位**：输出时间线清晰、时长可控的旁白文案，供分镜与语音合成使用。  
**前置依赖**：[SS0] 结构化数据、[SS2] `tone`、[SS6] 各环节时长配额。

## 精确落地流程

1. 按 [SS6] 中「生平回顾」环节时长估算字数（约 3 字/秒，可按语种调整）。
2. 采用「核心记忆 + 时间线」混合结构，避免流水账。
3. 每条 [SS0] 核心记忆至少分配约 15 秒等效时长。
4. 按 `[时间戳] 文本` 或文档约定的 `time` 区间输出，单段误差建议 ±1 秒内。
5. 在关键情感节点加入停顿标记（若下游 TTS 支持）。
